What is AngularJS

AngularJS is a full featured SPA framework and it is written in JavaScript. A single page application (SPA) is a web application or web site that fits on a single web page with the goal of providing a fast user experience similar to that of a desktop application. SPA can be developed using any frameworks like AngularJs, EmberJS, MeteorJs, ReactJs and so on.

AngularJS is a JavaScript MV structured framework to create dynamic web applications. AngularJS is open source, completely free and officially maintained by Google.
AngularJS is no different from plain HTML, but an extensions to HTML with new attributes i.e using ng-directives .


Before writing any code first we need to download and import AngularJS library on our webpage, then we extend html with angularjs directives.
We can also include AngularJS library from CDN [https://ajax.googleapis.com/ajax/libs/angularjs/1.6.5/angular.min.js].

AngularJS extends HTML with Directives.

In AngularJs with directive we can do DOM Manipulation, Data Binding and trigger Events. Directives teach HTML new tricks. Most important directive is ng-app which tells AngularJS that this part of webpage is AngularJS application

The AngularJS framework can be divided into following three major parts
  • ng-app : This directive use to define starting point of application and it will act as root element of an application.
  • ng-model : This directive binds the value of HTML controls to application data .
  • ng-bind : This directive binds the content of HTML element to application data.

Pros & Cons

Pros:
  • No page refresh : This mean to load data, we don't need to refresh whole webpage, just load specific part of our webpage which loads dynamic data.
  • Better user experience : SPA feels like a native application (like desktop app) very fast and responsive.
  • Two Way Data Binding : Data binding in AngularJS is the synchronization between the model and the view . This is one of the best feature of Angularjs. When data in the model changes, the view reflects the change, and when data in the view changes, the model is updated as well.
  • Ability to work offline: Even if user loses internet connection, SPA can still work because all the pages are already loaded.
  • Designed and Developed by Google : AngularJS is designed and built by bunch of dedicated Google engineers and now it officially maintained by Google.
Cons:
  • SEO : To index SPA app, search engine crawlers should able to execute javascript. One good news is now google and bing stared indexing ajax based webpages, but not all search engine started (Yahoo, AOL, Baidu, Yandex)
  • Confused to many ways to handle : One advantages with AngularJS is that it can be handled it in different ways, ironically this is the same feature which becomes a hurdle. As many beginner find this lil bit confusing.
